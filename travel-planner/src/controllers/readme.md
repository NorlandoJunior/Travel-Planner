# Travel Planner

## Overview

I am working to improve my programming skills by creating practical software projects and becoming more comfortable with modern development tools such as TypeScript, Git, GitHub, and Visual Studio Code. My goal is to continue learning software development concepts and building a strong foundation as a software engineer.

This software is a simple travel planning web application that allows users to create, view, edit, and delete trips. Users can manage information such as destination, travel dates, budget, notes, and trip status.

The purpose of creating this software was to practice TypeScript programming, object-oriented programming concepts, application architecture, DOM manipulation, and data management using local storage.

The application was developed using a structured architecture with models, services, controllers, and views to separate responsibilities and create cleaner, more maintainable code.
